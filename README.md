# 寿司郎自动预约

抢号工具

全自动抓取寿司郎（Sushiro）微信小程序预约，零外部依赖，纯 Go 标准库实现。

## 快速开始

```bash
# 一键启动（推荐）
./start.sh
```

就这么简单。脚本会自动编译并运行。

## 使用流程

```
1. 启动程序 → 自动设置系统代理
2. PC 微信打开寿司郎小程序 → 浏览门店、预约页面
3. 稉序自动捕获认证参数 → 选择门店 → 选择时段
4. 自动抢号 → 成功通知
```

## 首次运行

首次运行需要安装 CA 证书到 macOS 系统信任链，程序会提示输入 sudo 密码，只需一次。

## 详细步骤

### 1. 编译

```bash
go build -o sushiro .
```

### 2. 运行

```bash
./sushiro
```

### 3. 在 PC 微信中操作

- 打开微信 PC 版
- 搜索"寿司郎"小程序并打开
- 浏览门店列表（让程序捕获门店 ID）
- 进入预约页面，浏览时段（让程序捕获认证参数）

### 4. 选择门店和时段
程序会在控制台显示可用的门店和时段,按提示输入编号即可.

### 5. 自动抢号
选择完时段后程序自动开始轮询抢号.成功后发送 macOS 系统通知.

按 Ctrl+C 鳻统可随时退出,程序会自动清理系统代理.

## 工作原理

```
┌──────────┐     HTTPS (MITM)     ┌──────────────┐
│  PC 微信  │ ──────────────────→ │ 寿司郎服务器    │
│  小程序   │   ←── 本地代理 8080 ──→ │ crm-cn-prd.  │
└──────────┘     捕获认证参数      └──────────────┘
       │                                    │
       └── 捕获完成后，直连抢号 ──┘
```

1. 启动本地 HTTPS 代理 (MITM)
2. 设置 macOS 系统代理指向本地
3. 仅拦截寿司郎域名流量,其他流量直接放行
4. 捕获 X-App-Code, Authorization 等认证参数
5. 捕获完成后清理代理,直连 API 抢号

## 文件结构

```
main.go       入口，交互式流程
cert.go       CA 证书生成和管理
proxy.go      MITM 代理和参数捕获
api.go        HTTP API 客户端
config.go     配置加载
monitor.go    监控和抢号逻辑
notify.go     通知（飞书卡片）
slot.go       时段工具函数
state.go      状态持久化
```

## 注意事项

- 程序会修改 macOS 系统代理设置，退出时自动恢复
- CA 证书存储在 `~/.sushiro-proxy/`
- 状态文件存储在 `.sushiro_state.json`
- 抢号成功后程序自动退出
