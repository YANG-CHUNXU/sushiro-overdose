# sushiro-overdose 使用 FAQ

sushiro-overdose 是寿司郎排队/预约助手桌面工具。常见问题：

## 通行证（凭证）

**什么是通行证？**
只看排队和叫号预测不用登录。需要"动账号"的操作（预约、远程取号、取消、读我的单据）才会用到通行证——从寿司郎微信小程序请求里提取的一次凭证。

**怎么拿通行证？**
设置页点"拿通行证"：
- macOS：可先用"PC 微信自动抓"（推荐，最省事）
- Windows：用手机抓包导入（PC 微信抓不到）

**凭证为什么会失效？**
寿司郎一账号同时只认一个活跃会话——手机用过小程序就把电脑这边顶失效，反之亦然。失效后重新获取即可（约 3 分钟）。

**凭证失效的信号？**
官方接口返回 E010/error.server、401/403、取号/预约突然失败、手机重新打开过小程序。

## 叫号曲线 / 排队预测

**"按历史几点叫到几号"曲线怎么来的？**
来自线上数据库（Turso）的历史叫号聚合：按工作日/周末/节假日分类，每个时段的典型叫到几号（P50）、保守（慢，P20）、偏快（P80）。

**曲线是空的 / 不准？**
数据需要积累。线上采集器每 15 分钟采一次，confidence（置信度）从 low → medium → high 需要几周。本机采集更准但需自己开着采。

**ETA 预测（几点叫到我）准吗？**
优先用实时叫号速度（最准），融合历史规律。号码靠前/叫号快时最准；数据少时给保守区间。

## 通知

**快叫到了能提醒吗？**
能。设置页配通知渠道（飞书/Telegram/Bark/Server酱），快到你的号时推一条，不用一直盯着屏幕。

## 数据与隐私

- 排队和叫号信息本来就公开（小程序也显示），工具只读出来算
- 凭证只存自己电脑（~/.sushiro/），不上传第三方服务器
- 远程抢预约和取号走寿司郎官方接口，和手机操作一样
- 开源，代码可审计

## MCP 助手

**这个 MCP server 是干嘛的？**
装到 Claude Desktop/Cursor 后，AI 能帮你查排队数据、看你的预约状态、给到店建议、教你用工具。

**怎么开？**
sushiro 设置页 → "MCP 助手" → 开启。需要填 Turso 只读 token（去 Turso 控制台生成）。
