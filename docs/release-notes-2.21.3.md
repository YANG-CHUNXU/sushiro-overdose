# 2.21.3 Release Notes

2.21.3 是 GitHub 登录后线上 Turso 基准数据读取范围的小修版。

## 修复

- 高级曲线和趋势预测在用户选中门店时，只读取选中门店的线上基准数据；未选择门店时才读取全量基准。
- 多选门店时逐个请求单店基准并在本地合并，不再先拉取全量数据再过滤。
- 修复线上基准缓存未区分全量与单店的问题，避免单店读取被旧的全量缓存污染。
- 远端基准缓存改为按 key 保存，降低多选门店刷新时的重复 Worker/Turso 请求。

## 验证

- 新增 Cloudflare Worker 单店接口、缓存隔离、多选门店读取路径测试。
- 已验证 GitHub 登录后的线上基准读取会命中 `/api/queue/baseline/store?store_id=...`。
