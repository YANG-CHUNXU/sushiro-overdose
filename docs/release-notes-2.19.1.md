# Sushiro Overdose 2.19.1 Release Notes

2.19.1 是一个体验小版本，重点把本机持续采集从隐藏设置里前移到叫号曲线页。

## 主要变化

- **叫号曲线页新增本机持续采集开关**：用户在看“几点叫到几号”时，可以直接开启、暂停或立即采集一次。
- **采集价值说明前置**：页面明确说明本机采集会记录选定门店的叫号、等位和可预约时段，用来让曲线越用越准；数据只留在本机，不上传。
- **设置页降噪**：通知渠道、手机认证捕获、手动导入、移动端 UA、接口调试、本机诊断和日志默认折叠，只保留预约偏好常驻展示。
- **线上基准边界说明**：公开下载包不会内置作者 Turso 数据库 URL/token。普通用户默认不会接入作者线上库，后续应通过公开基准导出或整理好的数据模型安全消费。

## 不包含的变化

- 不把 Turso token 打包进公开 release。
- 不新增任何取号、取消预约或取消排队行为。
- 不改变现有采集数据落盘位置和 API 结构。
