# 2.21.2 Release Notes

2.21.2 是设置页信息架构与产品可用性小版本。

## 改进

- 设置页首屏新增关键状态总览：寿司郎认证、GitHub 线上基准、通知渠道、预测数据状态。
- 寿司郎认证前置，遇到 E010/error.server、401/403 或凭证异常时，引导用户优先重置并重新认证。
- 云端数据改成用户语言，不再默认暴露 Worker、Turso 等实现细节。
- 新增调试模式：手动导入凭证、移动端 UA、接口发现、云端服务地址等开发/排障工具默认隐藏。
- “数据收集”重命名为“预测准确度”，强调它用于提升“几点叫到、几点出发”的判断。
- “现在去吃”页面收敛为实时排队和自动取号计划；全国基准采集和旧版提醒配置归入调试模式。
- 多段到店提醒入口明确归到“我有号码”，提醒点属于临时规则，命中后自动删除，也可手动删除。

## 兼容性

- 不修改官方接口、抢号逻辑、通知配置格式和本地数据文件。
- 现有配置继续可用；本版本主要调整 Web UI 的信息架构、文案和默认可见性。
