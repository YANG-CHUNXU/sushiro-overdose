# Release notes: v3.1.2

3.1.2 做三件事：面板瘦身（删真正重复的控制面）、体验链路补强、寿司家族扩编 + 背景视觉。

## 变化

### 面板瘦身
- **设置 · 通行证**：删除「启动手机代理捕获 / 刷新捕获状态 / 停止捕获」面板——与向导里的「同 Wi-Fi 自动代理抓」完全同源重复，向导版有引导信号灯，保留它；设置页保留捕获状态诊断（指引回向导继续/停止）。
- **设置 · 预测准确度**：删除折叠顶部与下方 `sampleState` 重复的一行状态 chips（本机采集/凭证/线上基准在状态总览和详细 chips 里都有）。
- **我有号码**：门店是单选，chip 上的 ✕ 即清空，删掉冗余的「清空」按钮；顺带修复 ✕ 移除门店不写 localStorage、刷新后复活的问题。

### 体验链路
- **自动抢**：「添加目标 / 保存计划 / 启动蹲号」从页头移进「蹲还没放出的时段」盒内，操作贴着作用对象。
- **远程取号成功 / 取消排队号后**：立即刷新活跃票据，首页 ticket-hero 即时出现/消失，不用等 60 秒缓存。
- 侧栏偏好摘要的「未配置通知 → 去设置」改为直接定位到通知输入框（原先只跳设置页顶部）。

### 寿司家族扩编（8 → 12 只）🍣
- 新成员：**鲭鱼 saba**（青背波纹）、**章鱼 tako**（粉嫩吸盘）、**扇贝 hotate**（奶油棱线）、**海胆军舰 uni**（海苔裹金黄海胆，带小刺）。
- 全家福自动出现在：传送带、新手引导、通行证庆祝、各页随机吉祥物位。家福行支持换行，12 只在窄弹窗里不再溢出。

### 背景与主视觉
- **英雄区寿司游行**：右上角由单只三文鱼改为 三文鱼 → 海胆 → 章鱼 三只递进、透明度渐隐的小游行。
- **英雄区餐盘光晕**：右下角加一圈红金渐变的圆形暖光（纯 CSS，不挡内容）。
- **全局芝麻点纹理**：页面底色叠加极淡的红点纹理（26px 网格，纯 CSS 背景，无图片资源）。
- **传送带价位盘**：盘子边缘按寿司郎价位盘配色循环（白 → 金 → 红 → 银）。

- 12 只吉祥物 × 4 种表情经 QuickLook 实际渲染目检，画风统一、坐标无越界。
