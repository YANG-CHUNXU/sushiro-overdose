# 2.22.1 Release Notes

2.22.1 是每日 Routine 的认证安全修复版。

## 修复

- Routine 自动取号要求当天重新获取寿司郎凭证；没有当天有效凭证时进入 `needs_auth`，不会生成自动取号计划。
- 如果旧凭证今天已经通过“我的单据/基础接口自检”等凭证接口验证成功，也会视为今天可用。
- 如果已生成的 routine 计划后来发现凭证过期或不是当天凭证，会撤掉待执行计划，避免到点才失败。
- 临近预计取号时间且仍需认证时，会通过已配置通知渠道提醒用户刷新认证。

## 体验

- 顶部认证状态和设置页首屏会区分“今日可用”和“需今日刷新”。
- Routine 卡片新增“今天重新认证”入口，用户不用再去设置页找。
