# Release notes: v3.4.3

v3.4.3 是一个 UI 小修：修排队面板 / 排队趋势页文字溢出容器的问题，并统一输入框和胶囊组件的高度。

## 变化

### 文字溢出（qd 排队面板 / qt 排队趋势页）

- 排队面板和排队趋势页多处文字 / 数字会溢出容器：叫号大数字、KPI、胶囊标签、长店名在窄屏或长号码时会顶破卡片边界。
- 给所有 flex / grid 容器和卡片补 `min-width:0`（溢出根因：子项没这个属性时会被内容撑爆）。
- 叫号大数字 `.queue-call strong` 从固定 38px 改成 `clamp(26px, 7vw, 38px)` 并允许换行，老号码上千也不会顶破右侧折线图。
- KPI、胶囊、标签、`answer-chip`、`.ci`、长店名统一加 `overflow-wrap:anywhere`。
- 头部品牌 / 版本号在窄屏收敛。

### 输入框 / 胶囊组件统一

- `.dash-target` / `.tr` / `.sn-row` 内的输入框统一到 40px 高（之前 28 / 38 / 40 混着）。
- 开关 / 勾选 / 标签胶囊（`.switch` / `.check` / `.chip` / `.preset`，之前 34 / 40 / 34 混着）统一到 38px，与同行按钮视觉对齐。

## 验证

- `go build ./...` / `go test ./...` / 嵌入式 SPA 契约测试全绿。
- 移动端 390px 下用叫号 10888 + 长店名 + 180 分钟等待的压测数据渲染，修复后页面 0 个元素横向溢出（修复前必溢出）。
