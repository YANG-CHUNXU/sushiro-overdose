# Release notes: v3.5.0

v3.5.0 是一轮**产品动线重构**，基于多智能体并行调研（竞品/设计模式 + 现有动线诊断两份报告）。核心主线：把工具从「展示数据的面板」改成「给一个能照做的结论的决策器」。

调研依据：NN/G（设置页折叠 vs 展开、危险操作隔离、进度指示、渐进式披露）、Adobe/Balsamiq（一屏一主行动）、Wise/XE（双向换算交互）、海底捞/翠華排队类首屏范式。

## 关键变化

### 🎯 时间换算提升为一等公民（用户反馈最想要）

「几点取号 ⇄ 几点吃上」之前藏在「我有号码」页底部折叠里，还标了 `advanced-only`——简化版用户完全用不到，进阶版要滚到底。

- **提到首屏**：现在在 qd 页门店选择之后、实时压力之前，打开就看见。
- **双向用 ⇄ 换向按钮**（不再是下拉选「我想算」）：点圆形 ⇄ 按钮原地翻转方向，按钮 hover 时旋转 180°，方向记忆在本地。
- **输入即算**：去掉「算一下」按钮，填时间 / 切门店后 300ms 防抖自动算（参考 Wise 即时反馈）。
- **结果卡片**：主结论大字 + 区间 chips + 可折叠「为什么」依据。倒推方向（想几点吃→几点取号）额外提示「取号后前面可能被插队，实际等待 ±15 分钟，别当死线」。
- **对所有模式可见**：去掉 `advanced-only`，简化版也能用。

### 🔧 设置页重构（用户反馈「操作很迷」）

之前 7 个折叠全关、危险操作和日常配置混排、查看类（日志/洞察）塞进设置。按 NN/G 范式调整：

- **常用项默认展开**：寿司郎通行证、通知渠道现在进设置页就直接展开，不用先点头。
- **危险操作独立 Danger Zone**：停止进程 / 卸载清理从「安全与维护」折叠里的二级嵌套抽出来，单独放底部红框隔离区（NN/G 的 proximity 原则——危险操作不能挨着良性配置）。
- **文案与分组**：「安全与维护」改名「诊断与维护」（只留良性诊断）；引导文案说明「通行证和通知默认展开，诊断/日志/危险操作在下方」。
