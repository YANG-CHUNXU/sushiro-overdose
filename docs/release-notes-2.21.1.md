# 2.21.1 Release Notes

2.21.1 是 E010 凭证过期提示与认证重置小修版。

## 修复

- 官方返回 `E010/error.server` 时，明确提示“凭证已过期或被手机端登录顶掉”，引导用户先重置认证再重新获取凭证。
- 定时取号遇到 `E010/error.server` 时不再继续使用旧凭证重试；会停止未执行的自动取号计划并发送“需要重置认证”的通知。
- 新增本机寿司郎认证重置接口和设置页入口：停止当前自动操作、清除旧凭证、清内存 client、重置凭证健康状态，不会取消已经拿到的预约或排队号。
- 修复清除本地凭证后 Web 进程仍可能保留旧 API client 的状态同步问题。

## 注意

- 寿司郎凭证不是长期登录态，会过期，也可能被手机端再次打开小程序后顶掉；遇到 E010、401、403 或凭证异常时，请重新获取凭证。
