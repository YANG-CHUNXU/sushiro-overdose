# Release notes: v3.5.8

v3.5.8 回答两个问题：走势大图标出「几点叫到你」，历史规律区改成折线图展示「几点大概叫到多少号」。

## 变化

### 走势大图：标注「预计几点叫到你」

- 之前 ETA（预计叫到时间段）只藏在最后一个叫号点的 hover tooltip 里，图上看不到。
- 现在当用户填了当天排队号且有预测区间时，在 x 轴对应时段画**绿色半透明区间带** + 顶部标注「预计 12:18-12:42 叫到你（还差 100 桌）」。
- 一眼能看到大概几点叫到，不用悬停。

### 历史规律区：叫号折线图

- 之前历史规律区只有热力图 + weekday 卡片 + 趋势条，没有「几点叫到多少号」的直观折线。
- 新增「历史叫号曲线」折线图（放在历史规律区最前）：
  - x 轴 = 时段（10:00-22:00），y 轴 = 叫到号
  - **绿线 = 典型**叫到号、**蓝虚线 = 保守（偏慢）**、**黄虚线 = 偏快**
  - 悬停每个点看「历史典型叫到 X 号 / 保守-偏快 / 样本」
- 用户能直接读「按这家店的历史，12:00 大概叫到 ~400 号」，判断自己的号大概几点会被叫到。
- 数据来自 `called_curve`（本机采样 + 线上数据库基准）；没有数据时给出引导提示。

## 验证

- `gofmt` / `go build ./...` / `go vet` / `go test ./...` 全绿。
- CDP 真机回归（mock 数据）：ETA 区间带 + 标注渲染正确；历史叫号折线图 SVG/三色线/hover 热点全部正常。

> 注：ETA 区间和历史折线都需要叫号速率/采样数据。本地新装、没有本机历史或没连线上线数据库时，这些图会显示引导提示；用几天、积累采样后曲线会越来越准。
